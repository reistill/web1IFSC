Linha do Tempo da Internet e da Web

  1. Década de 1960: Surgimento das primeiras pesquisas sobre redes de computadores.
  2. 1969: Criação da ARPANET, considerada a precursora da Internet.
  3. 1983: Adoção do protocolo TCP/IP, permitindo a comunicação entre redes.
  4. 1984: Criação do sistema DNS (Domain Name System).
  5. 1989: Tim Berners-Lee, no CERN, propõe a criação da World Wide Web.
  6. 1990: Desenvolvimento do primeiro navegador e do primeiro servidor web.
  7. 1991: Disponibilização pública da World Wide Web (WWW).
  8. 1993: Popularização da Web com navegadores gráficos.
  9. Atualidade: Aplicações Web modernas com interação em tempo real e serviços em nuvem.

Imagens Históricas



Tim Berners-Lee e o CERN

Em 1989, Tim Berners-Lee propôs um sistema de compartilhamento de informações no CERN, que daria origem à World Wide Web.

"A Web foi criada para ser um espaço universal de compartilhamento de informações." — Tim Berners-Lee.

HTML, HTTP, Navegador e Servidor Web

HTML

O HTML (HyperText Markup Language) é a linguagem utilizada para estruturar páginas web.

HTTP

O HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre navegadores e servidores web.

Navegador

O navegador interpreta documentos HTML e apresenta o conteúdo ao usuário.

Servidor Web

O servidor web armazena páginas e responde às requisições feitas pelos navegadores.

Web 1.0 x Aplicações Web Modernas

A Web 1.0 era composta principalmente por páginas estáticas com pouca interação. Já as aplicações Web modernas oferecem recursos dinâmicos, comunicação em tempo real, armazenamento em nuvem e experiências semelhantes às de softwares desktop.

Lista de Definições

ARPANET
Rede criada pelo Departamento de Defesa dos EUA, considerada a origem da Internet.
TCP/IP
Conjunto de protocolos que permite a comunicação entre dispositivos conectados à Internet.
DNS
Sistema responsável por converter nomes de domínio em endereços IP.
HTML
Linguagem de marcação utilizada para estruturar páginas web.
HTTP
Protocolo utilizado para transferência de informações na Web.
