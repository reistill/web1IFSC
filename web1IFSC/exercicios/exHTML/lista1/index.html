Bem-vindo ao Portal Inicial de Programação para Internet

Este site foi desenvolvido como atividade da disciplina de Programação para Internet. Seu objetivo é apresentar conceitos fundamentais da Web, do protocolo HTTP, informações sobre estados brasileiros, currículo profissional e formas de contato.

Internet x Web

A Internet é a infraestrutura global formada por redes de computadores conectadas. A Web é um dos serviços que utiliza essa infraestrutura para disponibilizar páginas e conteúdos por meio de navegadores. Portanto, a Internet é a rede e a Web é um serviço que funciona sobre ela.
