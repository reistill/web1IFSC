Os elementos HTML possuem uma exibição ou display inline ou block

"Block - é uma exibição que inicia na margem esquerda até a direita, em que podemos determinar altura e largura do elementos com exibição block não dividem espaço na horizontal com outros elementos."

"Inline - é uma exibição que permite que o elemento seja apresentado dentro de uma linha com outros elementos como um elemento textual."
